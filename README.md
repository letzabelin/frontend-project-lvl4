# Аналог Slack-чата
## Упрощенная версия, но по-прежнему крутая:)

Проект выполнен в рамках прохождения профессии "Frontend JavaScript" на сайте [Хекслет](https://ru.hexlet.io/). Задача проекта — создать аналог [Slack](https://slack.com/) чата. Затрагиваемые темы:

* Websocket ([socket.io](https://socket.io/))
* REST API
* Authorization
* Authentication

* React ([hooks](https://reactjs.org/docs/hooks-faq))
* Redux ([reduxjs/toolkit](https://redux-toolkit.js.org/))
* Routing ([React Router](https://reactrouter.com/))
* Bundler ([Webpack](https://webpack.js.org/))
* Deploy ([Heroku](https://heroku.com/))
